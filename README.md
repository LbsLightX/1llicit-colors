# 1llicit-colors

> **A massive collection of color schemes for Termux.**

This repository serves as the theme engine for the **1llicit** project. It contains high-quality ports of the famous **Gogh** color schemes, optimized for Android.

## üé® Overview
- **Total Themes:** 377+
- **Source:** [Gogh-Co/Gogh](Gogh-Co/Gogh)
- **Format:** Termux `.properties`

## üì¶ Usage
These themes are designed to be installed natively via the **1llicit** dashboard.
Alternatively, you can manually apply any `.properties` file to `~/.termux/colors.properties`.
